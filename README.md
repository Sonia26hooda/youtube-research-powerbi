# 📊 YouTube Research & Strategy Dashboard (Power BI)

## 📌 Project Overview
This project demonstrates a **YouTube content analysis and strategy dashboard** built in Power BI.  
The goal is to analyze video performance, identify outliers, and uncover content strategies that drive higher engagement and views.  

The dataset is a **Sample of YouTube video metrics** across finance and productivity niches, including views, CTR, likes, comments, and subscriber growth.  

---

## 🎯 Key Features of the Dashboard
- **KPI Cards**  
  - Total Videos  
  - Outlier Count (videos > 2× average views)  
  - Average Views  
  - Average CTR  

- **Trend Analysis**  
  - Views over time (Upload_MonthYear)  
  - CTR trends by month  

- **Outlier Analysis**  
  - Viral videos spotted using Above_Avg_Ratio  
  - Scatter plot (CTR % vs Views)  

- **Engagement Insights**  
  - Likes & Comments analysis  
  - Subscriber growth over time  

- **Heatmap Matrix**  
  - Title Type × Thumbnail Type → Average Views  
  - Conditional formatting highlights **winning combinations** (e.g., *Numbers + Face+Emotion*).

## 🚀 Insights from the Dashboard
- **Face+Emotion thumbnails** with **Number-style titles** deliver up to **2× more views** than other combinations.  
- Outliers occurred mainly in **April and June**, showing strong seasonality in content performance.  
- Higher **CTR% strongly correlates with viral success**, proving the importance of thumbnails + hooks.  
- **Problem-Solution videos** had strong engagement (likes & comments), even when views were lower.
